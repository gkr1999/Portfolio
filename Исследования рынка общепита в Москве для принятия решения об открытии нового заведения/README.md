# Исследования рынка общепита в Москве для принятия решения об открытии нового заведения

## Задача проекта:
Исследование рынка общественного питания на основе открытых данных.

## Навыки и инструменты:
- Python
- Pandas
- Seaborn
- Plotly
- Визуализация данных

## Вывод:
Подготовила исследование рынка на основе открытых данных о заведениях общественного питания Москвы, визуализировала полученные данные. На основе данных выбрано место для открытия новой кофейни. В построении графиков использованы библиотеки seaborn и plotly.
